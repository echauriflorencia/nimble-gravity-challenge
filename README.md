# Nimble Gravity Challenge

This project is a React application built with Vite that allows a candidate to apply to available job positions using a provided API.

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- Fetch API

## Features

- Fetch candidate data by email
- Fetch available job positions
- Display job list with input field for GitHub repository URL
- Submit job application via POST request
- Proper handling of loading states and API errors
- Global feedback message for success and error responses
